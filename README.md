# kernel-from-scratch

A small educational operating system kernel written in C, built from scratch to understand low-level system internals.

## Status
This project is currently in its early stages and is still under development.  
The goal is to gradually implement core kernel components step by step.

## Purpose
This kernel is created purely for learning purposes, focusing on understanding how operating systems work internally rather than building a production-ready system.

## Scope
The project follows a structured path that includes:

- Boot process (Bootloader, GRUB)
- Low-level setup (GDT, IDT)
- Interrupt handling
- Basic input/output (VGA, Serial)
- Memory management (Paging, Allocation)
- Transition to user mode
- Simple file systems
- System calls
- Basic multitasking

## Technologies
- C (Primary language)
- x86 Architecture
- Assembly (for low-level integration)
- GRUB (Bootloader)
- QEMU / Bochs (Emulation)
